# Personal Portfolio Website

This repository contains the source code of my personal portfolio website.

The website introduces me and provides basic information such as my background, interests, and contact details. It is a simple and clean front end project created to practice web development fundamentals.

## About the Website

The homepage includes:

- A personal introduction section
- Basic details such as age, location, and interests
- A LinkedIn connection link
- An email input field with a Join button

This project focuses on structure, layout, and simple styling.

## Technologies Used

- HTML
- CSS
- Basic JavaScript

## Purpose

This project was built as my first portfolio website. It reflects my understanding of HTML structure, styling, and basic web layout design.

## How to Run

1. Clone the repository  
   git clone https://github.com/Priyanshu-Bisht-me/personal_site.git

2. Open the project folder

3. Open index.html in your browser

## Project Configuration Files

### package.json

This file contains the project metadata and dependency information. It defines the scripts used to run and build the project. For example, it allows commands such as npm run dev to start the development server and npm run build to create the production version of the website.

### vite.config.js

This file contains the configuration settings for Vite. It controls how the project is built and served during development. It ensures the website runs correctly using the Vite development environment.
